

### View Roles
```
./daoctl get proposals
```
#### Include proposals
```
./daoctl get roles --include-proposals
```
### View Assignments
```
./daoctl get payouts
```
### View Treasury
```
./daoctl get treasury
```
### View Details of a Ballot
```
./daoctl get ballot d4
```
